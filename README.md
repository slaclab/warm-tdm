# Warm TDM

*Current documentation can be found at:*

   https://slaclab.github.io/warm-tdm/

## Installation

From the root of the repository, run:

\```bash
bash scripts/install.sh
\```

This will create a conda environment called `warm-tdm-env` with all required dependencies,
and configure it to automatically set the `WARM_TDM_PATH` environment variable when activated.

To activate the environment:

\```bash
conda activate warm-tdm-env
\```
