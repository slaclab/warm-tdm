Group

Interface to a WarmTDM ‘Flange Group’ consisting of multiple Row and Column boards.
This class contains the top level configuration variables as well as a number of tuning and run control processes.

Processes:

  • SaOffsetProcess
  • SaTuneProcess
  • Sq1TuneProcess
  • FasTuneProcess
  • Sq1DiagProcess
  • TesRampProcess
